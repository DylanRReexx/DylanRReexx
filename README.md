- **Cloud Economics:** Reduced BigQuery operational costs by **70–90%** through refined ELT workflow optimization and lifecycle policy management.
- **Modern Data Stack:** Designed and deployed scalable data models using **Dataform**, enforcing version control and data quality standards for enterprise-level reporting.
- **AI/ML Pipeline:** Developed an end-to-end underwriting pipeline for vehicle abuse detection using **XGBoost**, turning raw sensor and claims data into actionable risk intelligence.
- **Large-scale Processing:** Managed data preparation and feature engineering for **50M+ records**, ensuring high-availability pipelines for continuous model training.
